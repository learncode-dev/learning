HTML - Imágenes y multimedia

Insertar imágenes

La etiqueta <img> se utiliza para insertar imágenes en una página web. A diferencia de otras etiquetas, <img> es una etiqueta auto-cerrada, es decir, no requiere una etiqueta de cierre. La imagen se especifica a través del atributo src, que indica la ubicación de la imagen, y el atributo alt, que proporciona una descripción de la imagen para usuarios con discapacidades visuales o si la imagen no se carga.


<img src="imagen.jpg" alt="Descripción de la imagen">

Atributos de las imágenes

Además de src y alt, las imágenes también pueden tener los atributos width y height, que permiten ajustar el tamaño de la imagen. Estos atributos son especialmente útiles si deseas cambiar el tamaño de una imagen sin modificar su archivo original. Es importante mantener la proporción entre el ancho y el alto para evitar que la imagen se distorsione.


<img src="imagen.jpg" alt="Descripción de la imagen" width="500" height="300">

Galerías básicas de imágenes

Puedes crear una galería básica de imágenes en HTML utilizando varias etiquetas <img> dentro de un contenedor <div>. Esto permite mostrar múltiples imágenes de manera organizada en una página web. Usualmente se acompañan de estilos CSS para agregar márgenes, bordes o crear un diseño de cuadrícula.


<div class="galeria">
  <img src="imagen1.jpg" alt="Imagen 1" width="250px">
  <img src="imagen2.jpg" alt="Imagen 2" width="250px">
  <img src="imagen3.jpg" alt="Imagen 3" width="250px">
</div>
