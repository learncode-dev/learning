HTML - Tablas y datos

Creación de tablas

Las tablas en HTML se crean utilizando la etiqueta <table>. Para definir las filas de la tabla, se utiliza la etiqueta <tr>, y cada celda dentro de una fila se define con la etiqueta <td>. Las tablas son útiles para mostrar datos organizados en filas y columnas, como una hoja de cálculo.


<table>
  <tr>
    <td>Celda 1</td>
    <td>Celda 2</td>
  </tr>
</table>
Celda 1 Celda 2




Encabezados de tablas

La etiqueta <th> se utiliza para definir las celdas de encabezado en una tabla. Estas celdas se muestran en negrita y centradas por defecto. Son útiles para identificar qué contiene cada columna o fila de la tabla.


<table>
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
  <tr>
    <td>Juan</td>
    <td>25</td>
  </tr>
</table>
Nombre Edad
Juan 25




Atributos de tablas

Los atributos border, cellspacing y cellpadding se usan para darle formato a las tablas.

  • border agrega un borde alrededor de la tabla y sus celdas.
  • cellspacing define el espacio entre las celdas.
  • cellpadding ajusta el espacio entre el contenido de las celdas y sus bordes.

<table border="2" cellspacing="5" cellpadding="10">
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
  <tr>
    <td>Juan</td>
    <td>25</td>
  </tr>
</table>
Nombre Edad
Juan 25




<thead>, <tbody> y <tfoot>

En una tabla HTML, las etiquetas <thead>, <tbody> y <tfoot> se utilizan para estructurar de manera semántica las diferentes partes de la tabla.


  • <thead>: Define el encabezado de la tabla, normalmente contiene las filas de encabezado (<th>).
  • <tbody>: Contiene el cuerpo principal de la tabla, donde se encuentran las filas con los datos de la tabla.
  • <tfoot>: Define el pie de la tabla, donde puedes colocar totales o resúmenes relacionados con los datos mostrados en la tabla.

Estas etiquetas no son obligatorias, pero ayudan a mejorar la accesibilidad y la organización del código, además de permitir que los navegadores las utilicen de manera eficiente para la renderización.

<table>
  <thead>
    <tr>
      <th>Nombre</th>
      <th>Edad</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Juan</td>
      <td>25</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Total</td>
      <td>1</td>
    </tr>
  </tfoot>
</table>
Nombre Edad
Juan 25
Total 1
