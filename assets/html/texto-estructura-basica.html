HTML - Texto y estructura básica

Encabezados: <h1> a <h6>

Los encabezados son etiquetas HTML utilizadas para definir títulos y subtítulos en una página web. Existen seis niveles de encabezados, desde <h1> (el más importante y grande) hasta <h6> (el más pequeño y menos destacado). Los motores de búsqueda utilizan estas etiquetas para entender la jerarquía del contenido en una página.


<h1>Título Principal</h1>
<h2>Subtítulo Nivel 2</h2>
<h3>Subtítulo Nivel 3</h3>
<h4>Subtítulo Nivel 4</h4>
<h5>Subtítulo Nivel 5</h5>
<h6>Subtítulo Nivel 6</h6>

Párrafos: <p>

La etiqueta <p> se utiliza para definir párrafos de texto. Es una forma básica de organizar contenido en bloques de texto que son fáciles de leer y estructurar.


<p>Este es un párrafo de ejemplo.</p>

Este es un párrafo de ejemplo.





Estilos (negrita, cursiva y subrayado): <b>, <i>, <u>

  • La etiqueta <b> se utiliza para destacar texto en negrita, lo que suele llamar más la atención.
  • La etiqueta <i> aplica cursiva al texto, ideal para enfatizar o diferenciar palabras o frases.
  • La etiqueta <u> subraya el texto, lo que puede ser útil para resaltar títulos, enlaces o palabras importantes en un contexto específico.

<p>Este texto está en <b>negrita</b>.</p>


<p>Palabra en <i>cursiva</i>.</p>


<p>Combinando <u>subrayado</u> y <b>negrita</b>.</p>

Este texto está en negrita.



Palabra en cursiva.



Combinando subrayado y negrita.





Listas no ordenadas: <ul>

  • La etiqueta <ul> define una lista no ordenada (con viñetas).
  • Los elementos de la lista se especifican dentro de etiquetas <li>.

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>


<ul>
  <li>Manzanas</li>
  <li>Naranjas</li>
</ul>
  • HTML
  • CSS


  • Manzanas
  • Naranjas




Listas ordenadas: <ol>

Las listas ordenadas son similares a las no ordenadas, pero se presentan con numeración en lugar de viñetas. La etiqueta <ol> define una lista ordenada, y los elementos de la lista se colocan dentro de etiquetas <li>. Estas son útiles cuando se necesita mostrar un orden específico en los elementos de la lista, como pasos de un proceso o clasificación.


<ol>
  <li>Primer paso</li>
  <li>Segundo paso</li>
</ol>


<ol>
  <li>Inicio</li>
  <li>Desarrollo</li>
  <li>Conclusión</li>
</ol>


<ol>
  <li>Primero prepara los ingredientes</li>
  <li>Luego cocina a fuego medio</li>
  <li>Finalmente sirve y disfruta</li>
</ol>
  1. Primer paso
  2. Segundo paso


  1. Inicio

  3. Conclusión

Elementos de bloque y en línea

Los elementos HTML se dividen en dos tipos principales según su comportamiento visual:

  • Los elementos de bloque ocupan todo el ancho disponible y comienzan en una nueva línea. Ejemplos comunes son <div>, <p>, <h1>, etc.
  • Los elementos en línea no rompen la estructura del documento y solo ocupan el espacio necesario para su contenido. Ejemplos de estos son <span>, <a>, <strong>, etc.

<div>
  Este es un bloque de texto.
</div>


<p>Un párrafo de texto.</p>
<span>Este es un texto en línea</span>


<div>
  Este bloque contiene <span>texto en línea</span> dentro de él.
</div>
Este es un bloque de texto.


Un párrafo de texto.

Este es un texto en línea

Este bloque contiene texto en línea dentro de él.
