HTML - Formateo de texto

Espaciado entre párrafos

El <br> es una etiqueta de salto de línea que se usa para crear un espacio entre las líneas de texto dentro de un párrafo, o para separar contenido en diferentes líneas sin agregar un nuevo párrafo.


<p>Este es un párrafo de texto.<br>
  Este es el siguiente párrafo en la misma línea pero con un salto de línea.</p>

Este es un párrafo de texto.

Bloques de texto

La etiqueta <blockquote> se usa para marcar citas largas o bloques de texto de otro origen. Generalmente, se le da un formato especial para diferenciarlo del resto del contenido.


<blockquote>
  <p>"El conocimiento es poder."</p>
  <pre>- Sir Francis Bacon</pre>
</blockquote>

"El conocimiento es poder."

- Sir Francis Bacon




Citas y referencias

La etiqueta <q> se usa para marcar citas más cortas dentro del texto, que generalmente se muestran con comillas.


<p>Albert Einstein dijo: <q>La imaginación es más importante que el conocimiento.</q></p>

Albert Einstein dijo: La imaginación es más importante que el conocimiento.





Uso del <code>

La etiqueta <code> se usa para mostrar fragmentos de código o comandos dentro de un bloque de texto.


<p>Para crear un enlace, usamos el siguiente código: <code><a href="enlace.html">Click aquí</a></code></p>
