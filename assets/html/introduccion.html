HTML - Introducción

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para estructurar contenido en la web. Se basa en el uso de etiquetas para definir diferentes elementos como texto, imágenes y enlaces.


<!DOCTYPE html>
<html>
  <head>
    <title> Mi Página Web </title>
  </head>
  <body>
    <h1> ¡Hola Mundo! </h1>
    <p> Esta es mi primera página. </p>
  </body>
</html>

¡Hola Mundo!

Esta es mi primera página.





Estructura básica de un documento HTML

Un documento HTML siempre comienza con <!DOCTYPE html> seguido de las etiquetas <html>, <head>, y <body>.


<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Página Ejemplo</title>
  </head>
  <body>
    <h1>Bienvenido</h1>
    <p>Aquí va el contenido principal.</p>
  </body>
</html>

Bienvenido

Aquí va el contenido principal.





Etiquetas HTML: ¿Qué son y cómo se usan?

Las etiquetas son los bloques fundamentales en HTML. Cada etiqueta comienza con un nombre rodeado por < y >. Ejemplo: <p> para párrafos o <h1> para encabezados.


<h1> Este es un encabezado principal. </h1>


<a href="https://learncode.es/" > Haz clic aquí </a>


<img src="imagen.jpg" alt="Descripción" />

Este es un encabezado principal.



Haz clic aquí





Atributos HTML

Los atributos añaden propiedades a las etiquetas HTML, como definir enlaces con href o imágenes con src.


<img src="photo.png" alt="Descripción breve" />


<a href="https://google.com" target="_blank" > Ir a Google </a>


<p title="Información adicional" > Este párrafo tiene un atributo "title". </p>
