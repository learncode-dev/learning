HTML - Comentarios y Metadatos

Comentarios

Los comentarios en HTML son fragmentos de texto que no se muestran en la página web pero sirven para explicar el código a los desarrolladores o para dejar notas dentro del documento.


<!-- Este es un comentario -->
  <p>Este texto se muestra en la página.</p>
<!-- Otro comentario -->

Este texto se muestra en la página.





Metadatos

  • <meta>: Se usa para definir metadatos que no se muestran en la página, como el tipo de contenido o la descripción de la página.
  • <title>: Define el título de la página, que aparece en la pestaña del navegador.

<head>
  <meta charset="UTF-8">
  <meta name="description" content="Descripción de la página web">
  <title>Mi Página Web</title>
</head>




Optimización en buscadores con <title>

El <title> es crucial para la optimización en motores de búsqueda (SEO), ya que los motores de búsqueda lo usan para determinar de qué trata la página. Un título claro y relevante puede mejorar la visibilidad en los resultados de búsqueda.


<head>
  <title>Guía de HTML y CSS</title>
</head>
