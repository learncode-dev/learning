Enlaces y atributo "href"

Los enlaces son una parte fundamental de la navegación en la web. Se crean utilizando la etiqueta <a>, que se utiliza para definir un hipervínculo. El atributo href se utiliza dentro de la etiqueta <a> para especificar la URL del destino del enlace. Cuando un usuario hace clic en el enlace, el navegador lo lleva a la URL indicada.


<a href="https://www.learncode.es">Visita nuestra página web</a>


<a href="https://www.google.com">Ir a Google</a>


<a href="https://www.facebook.com">Visitar Facebook</a>

Enlaces internos y externos

  • Enlaces internos: Estos son enlaces que apuntan a otras partes dentro de la misma página o sitio web. Se usan comúnmente con anclajes o identificadores en la página. Para hacer un enlace interno, simplemente coloca el símbolo # seguido del id del elemento de destino.
  • Enlaces externos: Son enlaces que llevan a otra página fuera de tu sitio web. Esto se especifica con una URL completa que contiene http:// o https://.

Enlaces de correo electrónico y teléfonos

  • El atributo mailto: en un enlace permite a los usuarios enviar un correo electrónico directamente desde la web. El valor del atributo href comienza con mailto: seguido de la dirección de correo electrónico del destinatario. Esto abre automáticamente el cliente de correo del usuario con la dirección pre-rellenada.
  • El atributo tel: en un enlace permite a los usuarios realizar una llamada telefónica directamente desde la web. El valor del atributo href comienza con tel: seguido del número de teléfono al que se desea llamar. Al hacer clic en el enlace, se abrirá la aplicación de teléfono del usuario para realizar la llamada.

<a href="mailto:info@ejemplo.com">Enviar un correo</a>


<a href="tel:+34678234736">Llamar</a>

Enlaces con anclaje (#)

Los anclajes son un tipo especial de enlace interno que permiten navegar hacia una parte específica de la misma página. Se utiliza el atributo href con el símbolo # seguido de un id que debe coincidir con el de un elemento en la página. Es útil para crear menús de navegación o saltar directamente a secciones específicas de contenido largo.
