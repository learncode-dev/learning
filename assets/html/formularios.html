HTML - Formularios

Formularios (<form>)

La etiqueta <form> se usa para crear un formulario donde los usuarios pueden ingresar datos. El formulario puede contener diferentes tipos de campos, como cajas de texto, botones y casillas de verificación.
Los atributos action y method especifican el destino y el tipo de envío de los datos, respectivamente.


<form action="procesar_datos.php" method="post">
  <!-- Aquí irán los campos del formulario -->
</form>




Campos de texto (<input>)

El campo de texto <input> permite a los usuarios ingresar información, como su nombre, correo electrónico, etc. Los atributos permiten especificar el tipo de entrada.


<form>
  Nombre: <input type="text" name="nombre">
</form>

Botones

El <button> es una etiqueta que permite crear botones dentro de un formulario, que pueden realizar acciones al ser presionados.


<form>
  <button type="submit">Enviar</button>
</form>




Áreas de texto

La etiqueta <textarea> se utiliza para crear un área de texto más grande, donde los usuarios pueden escribir múltiples líneas de texto.


<form>
  Comentarios: <textarea name="comentarios" rows="4" cols="50"></textarea>
</form>
Comentarios:




Checkbox y radio

  • <input>type="checkbox">: Permite seleccionar una o más opciones.
  • <input>type="radio">: Permite seleccionar solo una opción de un conjunto de opciones.

<form>
  <input type="checkbox" name="suscripcion" value="newsletter"> Suscribirse al boletín
  <input type="radio" name="opcion" value="si"> Sí
  <input type="radio" name="opcion" value="no"> No
</form>
Suscribirse al boletín Sí No




Labels

La etiqueta <label> se usa para asociar una etiqueta descriptiva con un campo de formulario, mejorando la accesibilidad.


<form>
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre">
</form>




Envío de datos

El atributo action en la etiqueta <form> define la URL a la que se enviarán los datos del formulario cuando se envíe.


<form action="procesar_datos.php" method="post">
  <button type="submit">Enviar</button>
</form>
